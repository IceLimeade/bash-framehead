# `hash::sha3_256`

SHA3-256 hash of a string

## Usage

```bash
hash::sha3_256 ...
```

## Source

```bash
hash::sha3_256() {
    if runtime::has_command openssl; then
        _hash::pipe "$1" openssl dgst -sha3-256 2>/dev/null | awk '{print $NF}'
    else
        echo "hash::sha3_256: requires openssl with sha3 support" >&2
        return 1
    fi
}
```
