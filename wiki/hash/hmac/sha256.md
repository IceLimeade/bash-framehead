# `hash::hmac::sha256`

HMAC-SHA256

## Usage

```bash
hash::hmac::sha256 ...
```

## Source

```bash
hash::hmac::sha256() {
    local key="$1" msg="$2"
    if runtime::has_command openssl; then
        printf '%s' "$msg" | \
            openssl dgst -sha256 -hmac "$key" 2>/dev/null | awk '{print $NF}'
    else
        echo "hash::hmac::sha256: requires openssl" >&2
        return 1
    fi
}
```
