# `hash::hmac::sha512`

HMAC-SHA512

## Usage

```bash
hash::hmac::sha512 ...
```

## Source

```bash
hash::hmac::sha512() {
    local key="$1" msg="$2"
    if runtime::has_command openssl; then
        printf '%s' "$msg" | \
            openssl dgst -sha512 -hmac "$key" 2>/dev/null | awk '{print $NF}'
    else
        echo "hash::hmac::sha512: requires openssl" >&2
        return 1
    fi
}
```
